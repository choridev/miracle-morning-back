# Miracle Morning - Back-end

## 프로젝트 소개

'미라클 모닝'은 사용자들이 아침 루틴을 설정하고 실천하며, 이를 통해 하루를 활기차게 시작할 수 있도록 돕는 자기 계발 애플리케이션입니다. 이 프로젝트는 해당 애플리케이션의 백엔드 서버로, 사용자 관리, 루틴 설정, 결과 기록, 통계 등 핵심 비즈니스 로직을 처리합니다.

## 주요 기능

- **사용자 관리**: JWT 기반의 인증/인가 시스템을 통해 안전하게 사용자를 관리합니다.
- **루틴 관리**: 사용자는 자신만의 '미라클 모닝' 루틴을 생성, 조회, 수정, 삭제할 수 있습니다.
- **실천 기록**: 매일 루틴 실천 여부를 기록하고 관리할 수 있습니다.
- **통계**: 주간, 월간 루틴 달성률과 같은 통계 데이터를 제공하여 동기를 부여합니다.

## 기술 스택

- **언어**: Java 17
- **프레임워크**: Spring Boot 3.2.3
- **데이터베이스**: MySQL 8.0
- **인증**: Spring Security, JWT (JSON Web Token)
- **캐시/세션**: Redis
- **빌드 도구**: Gradle
- **컨테이너**: Docker, Docker Compose
